# BM_com_v2 (บ้านมีคอม V2)

เว็บไซต์ร้านขายเซ็ตคอมพิวเตอร์ประกอบสำเร็จรูป "คอมพี่ประกอบแล้ว ไม่ต้องจัดสเปคเอง" มีทั้งหน้าร้านสำหรับลูกค้าและระบบหลังบ้านสำหรับแอดมิน

## ฟีเจอร์ที่ใช้งานได้จริงตอนนี้

- ลูกค้าดูรายการสินค้า ใส่ตะกร้า และปรับจำนวนสินค้าได้ โดยไม่ต้องเข้าสู่ระบบ
- สมัครสมาชิกและเข้าสู่ระบบด้วยอีเมลและรหัสผ่าน
- แอดมินเพิ่ม แก้ไข และลบข้อมูลสินค้าได้ครบวงจร พร้อมดูแดชบอร์ดสรุปยอดขาย จำนวนคำสั่งซื้อ และจำนวนสมาชิก

ฟีเจอร์ที่ยังอยู่ระหว่างวางแผนพัฒนา (เช่น การยืนยันสั่งซื้อจริง การชำระเงินผ่าน QR พร้อมเพย์ และการตรวจสลิปอัตโนมัติ) ดูรายละเอียดได้ใน [docs/analysis-and-design.md](./docs/analysis-and-design.md)

## เริ่มต้นใช้งานโปรเจกต์

```bash
cd frontend
npm install
npm run dev
```

จำเป็นต้องมีไฟล์ `.env` ในโฟลเดอร์ `frontend` พร้อมค่า `VITE_SUPABASE_URL` และ `VITE_SUPABASE_ANON_KEY` ของโปรเจกต์ Supabase ที่ใช้งาน (ดูตัวอย่างค่าที่ต้องตั้งได้จากผู้ดูแลระบบฐานข้อมูล)
