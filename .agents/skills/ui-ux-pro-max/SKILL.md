# UI/UX Pro Max

Use this skill whenever the user requests frontend redesign, UI polish, landing page improvements, or modern visual system work.

## Design Principles
- Prioritize clarity, hierarchy, and visual comfort.
- Use a light, premium, polished aesthetic with generous spacing.
- Favor soft shadows, subtle gradients, rounded surfaces, and smooth transitions.
- Keep interactions responsive with hover, focus, and motion states.
- Ensure strong contrast and accessible typography.
- Use modern iconography and clean component structure.

## Recommended Visual Direction for This Project
For this website, use a premium e-commerce / computer shop aesthetic with:
- A strong red accent color for branding and CTAs.
- Clean white or soft gray surfaces for cards and sections.
- Rounded panels, subtle elevation, and polished shadows.
- Clear hero section, featured products, trust badges, and concise CTAs.
- Responsive layouts for mobile, tablet, and desktop.

## Frontend Implementation Guidance
- Prefer Tailwind-style utility classes for rapid, consistent UI changes.
- Use large headings, balanced spacing, subtle cards, and strong content hierarchy.
- Keep buttons prominent and accessible.
- Improve empty states, navigation, and product presentation.
- Add micro-interactions such as hover transitions and focus rings.
- Maintain the existing product-oriented tone while elevating visual quality.

## Component Patterns
- Hero section with headline, subtext, CTA, and visual support.
- Feature cards with icons and short descriptions.
- Product cards with image, price, short summary, and CTA.
- Testimonial or trust section for credibility.
- Checkout and profile forms with clean field styling.

## Constraints
- Do not use harsh neon colors or excessive animation.
- Avoid cluttered layouts.
- Respect the existing brand of BM Computer while modernizing it.

## When to Apply
Apply this skill for tasks such as:
- redesigning the homepage
- improving the cart and checkout experience
- polishing admin and profile screens
- refining navigation and mobile layout
- updating forms and empty states
